# Agent Rules

- All comments, names, and documentation must be in English.
- Keep the layout stable: reuse existing classes, CSS variables, spacing, and DOM hooks unless a requested change requires otherwise.
- Split UI into small React components by responsibility.
- Move reusable pure logic into `src/utils`.
- Add or update matching unit tests in `tests`, mirroring the `src` path.
- Keep API calls and GitHub token handling on the Node server.
- Avoid broad rewrites outside the requested area.
